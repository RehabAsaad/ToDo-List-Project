To-Do List Application

Description

This is a simple To-Do List application written in C++ that allows users to manage tasks. It supports adding, deleting, displaying, editing, and marking tasks as completed. It offers a text-based interface and works in the command line.

Features

Add Task: Add new tasks with a name, description, and due date.
Delete Task: Delete a specific task from the list.
Display Tasks: View all tasks, their descriptions, due dates, and completion status.
Mark Task as Completed: Mark a task as completed.
Edit Task: Modify the name, description, or due date of an existing task.
